Overview

This skill provides capabilities to retrieve all tasks or filtered tasks from a ToDo List app.

Purpose

This skill allows Claude to:

Fetch all tasks in the system

Filter tasks by status (completed, pending)

Return structured JSON with task details

Quick Start

When a user wants to list tasks:

Receive optional filter parameters

Retrieve matching tasks from the database

Return tasks as JSON array

Core Capabilities
Task Retrieval

List all tasks

Filter by status, due date, or other parameters

Include ID, title, description, due date, and status

Response Management

Return JSON array of tasks

Handle empty results gracefully

Usage Examples

"List all tasks"

"Show only completed tasks"

"Return tasks due this week"
